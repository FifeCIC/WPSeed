## Tips for Best Screenshots

1. **Use clean test data** - No Lorem Ipsum or test123
2. **Full browser width** - Show full interface
3. **Hide personal info** - No real API keys visible
4. **Good lighting** - Clear, readable text
5. **Consistent theme** - Use default WordPress admin theme
6. **PNG format** - Better quality than JPG
7. **Optimize file size** - Keep under 1MB each

## After Capturing

1. Save as `screenshot-1.png`, `screenshot-2.png`, etc.
2. Place in `assets/screenshots/` directory
3. Verify dimensions (1280x720 minimum)
4. Check file sizes (under 1MB recommended)
5. Test display in readme.txt preview
